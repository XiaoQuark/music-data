# Music Data Project

A JavaScript application that analyses music listening data and displays statistics for a selected user.

## Features

For each user, the application displays:

- Most listened song by number of listens
- Most listened song by listening time
- Most listened artist by number of listens
- Most listened artist by listening time
- Most listened song on Friday nights (5pm to 4am)
- Longest listening streak
- Songs listened to on every listening day
- Top genres by number of listens

Questions that do not apply to a user are automatically hidden.

## Running locally

Serve the project over HTTP and open it in a browser.

Example using http-server:

```bash
npx http-server
```

## Running tests

```bash
npm test
```

## Accessibility

The application achieves a Lighthouse accessibility score of 100.
